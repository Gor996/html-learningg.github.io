URL, WWW, HTTP

Դիտարկված օրինակներում, որպես հղման հասցե օգտագործվում էր փաստաթղթի անունը: Ընդհանրապես կարելի է կիրառել URL-հասցեն (Unifգrm Resգurce Locater - միանման ռեսուրսի ցուցադրող): URL-հասցեի ֆորմատը ներառում է իր մեջ ցանցային ծառայության տեսակը (կապի պրատակոլը), սերվերի հասցեն, փնտրման ճանապարհը և փաստաթղթի անվանումը:

<a href="http://google.com">Հղում դեպի URL հասցե</a>

Եթե օգտագործում ենք հասցե, որը սկսվում է http-ից, ապա այդ այդ ռեսուրսի հասանելիությունը ապահովվում է HTTP ծառայության պրատակոլը (Hypertext Transfer Protocol - հղումների փոխանցման պրատակոլը): Այս պրատակոլը World Wide Web -ում HTML փաստաթղթերի ինֆորմացիայի փոխանակման ժամանակ օգտագործվում է որպես հիմնական:

Եթե հղման հասցեից առաջ նշվում է mailto, դա նշանակում է, որ անհրաժեշտ է օգտագործել պրատակոլը հաղորդագրության փոխանցման, էլեկտրոնային փոստով եղանակը:

<a href="mailto:anun@mail.ru">Հղում դեպի էլ. փոստի հասցե</a>
