Բացատրական բառարան

Address - Կոնկրետ ցանցի ֆայլում գտնվող տեղեկատվությամբ յուրօրինակ կոդ:

 

Anchor (<a ... >) - Այն, ինչը հենց ձևակերպում է հիպերտեքստային հղումը:

Attribute - Տարրի պարամետր կամ հատկություն: Ըստ էության սա մի փոփոխական է, որն ունի միօրինակ անուն և որին կարող է վերագրվել արժեքների որոշակի հավաքանի: Ենթադրվում է, որ ատրիբուտի նշանային արժեքները դրվում են չակերտների մեջ, բայց որոշ բրոուզերներ թույլ են տալիս չօգտագործել չակերտներ:

Browser կամ բրոուզեր - Հաճախորդի համակարգչի վրա աշխատող ծրագիր, որը HTTP արձանագրության շրջանակներում ուղարկում, ընդունում և արտապատկերում է ֆայլերը£ Ժամանակակից բրոուզերները հասկանում են նաև այլ արձանագրություններ, օրինակ` FTP կամ Gopher, Ներկայումս ամենատարածված բրոուզերներից են Netscape, Mozzile Firefox, Internet Explorer, Mosaic և Opera: 

 

Font - Տառատեսակ կամ յուրօրինակ ոճի նշանների ընտանիք:

 

Frame - 1. Փաստաթղթի տիրույթ` իր վերելակով (scroll),

2. Բարդ “շարժուն” գրաֆիկական ֆայլի մի առանձին պատկեր 

 

GIF (Graphic Interchange Format) - Ճկուն, բայց ոչ բարձրորակ պատկերների գրաֆիկական ձևաչափ: WWW-ում կիրառվում է հիմնականում ոչ մեծ գրաֆիկայի, նշանների, պիկտոգրամների, ինչպես նաև ոչ մեծ թվով գույներով պատկերների դեպքում: Լուսանկարչական որակի պատկերների համար խորհուրդ է տրվում JPEG ձևաչափը:

 

HTML - Հիպերտեքստային փաստաթուղթն իր նախնական տեսքով, երբ տեսանելի են բոլոր տարրերն ու ատրիբուտները:

 

HTTP (HyperText Transport Protocol) - Ինտերնետ արձանագրություն, որն օգտագործվում է WWW-ում HTML փաստաթղթերի փոխանակման համար:

 

Hyperlink - Գերկապ (հղում): Տեքստի հատված կամ գրաֆիկայի տարր HTML էջի վրա, որը հղում է այլ տեղ նույն փաստաթղթի վրա կամ այլ փաստաթղթի օբյեկտի վրահնարավորէդասավորված այլ սերվերի վրա:

 

Internet - Համաշխարհային համակարգչային ցանց, որը ներառում է տասնյակ հազարավոր տեղական ցանցեր` մի քանի տասնյակ միլիոն համակարգիչներով: IP Address - ինտերնետարձանագրության 32 բիթանի հասցե, որը ներառում է հանգույցի և ցանցի համարը: Դա 0-ից 254 մեծության չորս թվերի համախումբ է, իրարիցբաժանվածկետերով: Դոմենների անունների և IP հասցեների միջև գոյություն ունի փոխմիարժեք համապատասխանություն:

 

JPEG (Joint Photographic Experts Group) - Գրաֆիկական ձևաչափ, որն ընդունված է որպես ստանդարտ ձևաչափ Web էջերի ձևավորման համար:

 

Marquee - Վազող տող HTML փաստաթղթում:

 

MPEG (Moving Pictures Expert Group) - Արձանագրություն, որի միջոցով փաթեթավորվում է տեսագրությունը:

 

Page – Էջ: World Wide Web ցանկացած փաստաթուղթ:

 

Script - Web էջի կազմում մտցված ծրագիր, դրա հնարավորությունների ընդլայնման համար:

 

 

Tag - Տարրի սկզբնական կամ վերջնական ցուցիչ: Տեգերը որոշում են տարրերի գործողության սահմանները և միմյանցից անջատում են տարրերին:

 

 

URL (Universal Resource Locator) - Հասցե ինտերնետում, որը միանշանակ նույնացնում է էջը տվյալների հետ: Այն կազմված է արձանագրության, դոմենի, ճանապարհիևֆայլիանվան նշանակումից:

World Wide Web, WWW կամ պարզապես Web - Համաշխարհային սարդոստայն, հիպերտեքստային փաստաթղթերի մատչելիության բաշխված համակարգ ինտերնետում:

Web էջ - Հիպերտեքստի ձևաչափով պատրաստված և World Wide Web-ի մեջ տեղադրված փաստաթուղթ “ֆայլ”:
