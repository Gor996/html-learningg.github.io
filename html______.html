HTML փաստաթղթի կառուցվածքը, գրելու կարգը և պահպանումը

HTML (Hyper Text Markup Language), Tag, Browser

HTML - նշանակում է Hyper Text Markup Language, որը համարվում է վեբ ծրագրավորման լեզու և նախատեսված է ինտերնետային էջերի նկարագրության համար:

Ինտերնետային էջի նկարագրությունը պահպանվում է HTML կոդերում, որոնք իրենց հերթին պահպանվում են սովորական տեքստային փաստաթղթերում` HTM կամ HTML ընդլայնումներով: Երբեմն այդ ծրագրերը անվանում են HTML փաստաթղթեր, բայց սովորաբար HTML փաստաթուղթ համարվում է այն, ինչը հնարավոր է տեսնել բրոուզերի պատուհանի միջոցով: Web բրոուզերի նպատակն է (օրինակ Internet Explorer-ի կամ Firefox-ի) ընթերցել HTML փաստաթղթերը ու ցուցադրել այդ փաստաթղթի տեգերը ինտերնետային էջերի տեսքով: Բրոուզերը չի ցուցադրում HTML տեգերը, բայց օգտագործում է դրանք էջի բովանդակության մեկնաբանության համար:

Tag (տեգ) - Տարրի սկզբնական կամ վերջնական ցուցիչ: Տեգերը որոշում են տարրերի գործողության սահմանները և միմյանցից անջատում են տարրերին:

Browser կամ բրոուզեր - Հաճախորդի համակարգչի վրա աշխատող ծրագիր, որը HTTP արձանագրության շրջանակներում ուղարկում, ընդունում և արտապատկերում է ֆայլերը£ Ժամանակակից բրոուզերները հասկանում են նաև այլ արձանագրություններ, օրինակ` FTP կամ Gopher, Ներկայումս ամենատարածված բրոուզերներից են Netscape, Mozzile Firefox, Internet Explorer, Mosaic և Ope

Որպես կանոն, տեգերը գրվում են մեծատառ, բայց ժամանակակից բրոուզերներից մեծամասնությունը, թույլատրվում են տեգերի գրառումը և' մեծատառ և' փոքրատառ: Բոլոր տեգերը իրենցից ներկայացնում են անգլերեն բառերի կրճատումները: Օրինակ IMG - image (նկար) բառի կարճ տարբերակն է:

HTML փաստաթղթի ծրագիրը պարտավոր է սկսվել <HTML> տեգից, և ավարտվել </HTML> տեգով: Նրանց միջև գտնվում են ծրագրի մնացած տեգերը, կամ տեքստը, այն ինչ պատրաստվում է ցուցադրել բրոուզերի պատուհանում: Որոշ տեգեր օգտագործվում են զույգերով (օրինակ ` <HTML> և </HTML>): Նրանցից առաջինը համարվում է բացվող, իսկ երկրորդը փակվող: Երբեմն զույգերով տեգերը անվանվում են տարրային, քանի որ նրանց միջև կարելի է տեղադրել այլ տեգեր: Տարրային տեգերում կարելի է տեղադրել այլ տեգեր, այդ թվում նաև տարրային, այսինքն տեգերը կարող են լինել ներառված: Գոյություն ունեն նաև միայնակ տեգեր: Նմանատիպ տեգերի համար չկան համապատասխան փակող տեգեր: Միայնակ տեգերից կարելի է նշել <P> տեգը (Պարբերություն):

Տեգերը կարող են պարունակել հատկություններ, որոնք կոչվում են հատկանիշներ, որոնք իրենց հերթին կարող ունենալ նաև փաստարկային արժեքներ: Տեգերը ծրագրում համարում են հրամաններ, հատկանիշը - հատկություն, իսկ փաստարկները - հատկության արժեքը:
