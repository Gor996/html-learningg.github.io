CSS - (Cascading Style Sheets) ոճերի կասկադային աղյուսակներ: Սահմանումը, կիրառման վայրը

CLASS

Մենք կարող ենք ստեղծել ոճերի աղյուսակ, նրանց անվանում ամրացնելով: Այդ անունը նշանակվում է ինչպես սովորական անվանում, բայց կետիկով (պիկսելով) առաջին սիմվոլի կարգավիճակում: Այդ դեպքում տեգերում մենք կարող ենք դիմել տվյալ աղուսակին նրա անվանումով, օգտագործելով բնորոշ հատկանիշ CLASS=ոճի_անունը, որտեղ ոճի անվանումը օգտագործվում է արդեն առանց կետիկի (պիկսելի):

Ոճավորում class-ի օգնությամբ՝

<STYLE>

.classi_anun {COLOR:red;size:12px;}

</STYLE>

Իսկ class-ը արդեն օգտագործում ենք մեր անհրաժեշտ վայրում՝

<div CLASS=”classi_anun”></div>
